Digital Logic

Lucas Xu / 2018-03-07

Computer language

Computer ( or say, CPU in computer) computes using 0s and 1s at a really fast rate. In other words, it uses a Binary System. (base-2 numeral system)

In computer term, we group binary into 8 numbers or bits. Technically, a bit is a binary digit. Historically, we used 8 bits because in the early days of computing, hardware utilized the base-2 numeral system to move bits around. 2 to the 8th numbers offered us a large enough range of values to do the computing we needed.

Therefore, a group of 8 bits is referred to as a Byte. Each byte can store one character, and we can have 256 possible values. Everything including a video, an image, text or anything else is 0s and 1s.

For example:

01001100    L
01110101    u
01100011    c
01100001    a
01110011    s

Extensive knowledge: we also have some other bases such as base 8 (Octal) and base 16 (Hexadecimal).

Character Encoding

Character encoding is used to assign the binary values to characters so that they become human readable. It works like a dictionary for a computer to look up which human characters should be represented by a given binary value.

The oldest character encoding standard used is ASCII. The great thing about ASCII is we only use 127 values out of possible 256. Obviously, it was not enough. (For example, we need to encoding Chinese characters as well)

The most prevalent encoding standard we used today is called UTF 8. It stands for Unicode Transform Format 8-bit. Along with having the same ASCII table, it also lets us use a variable number of bytes. UTF 8 allows us to store a character in more than one byte. It is based on Unicode Standard helps us represent character encoding in a consistent manner.

Binary

Binary in today’s computer isn’t done by reading holes like it previously did. It uses electricity via transistors allowing electrical signals to pass through. But just having transistors isn’t enough for our computer to be able do complex tasks. Then we have logic gates. Through a compiler, a computer can turn human-readable instructions into 0s and 1s.
